I learned the basic design of web pages. But I realized that I needed to learn any programming language to set up my own servers, with databases, login, and other advanced features. I've always liked using the latest technology. So look for the most practical language, I knew the Firebase platform that facilitated me Hosting my first pages, and I started a course of HTML5 and CSS3.

I was enrolled in a Course call "Build Responsive Real World Website with HTML5 and CSS3 by Jonas Schmedtmann" in Udemy for USD 10. With this course, I started building my first projects with 100% code without Sketch or any other app. This was my first project in CSS3 and Httm5, it a Website for a Restaurant or Food Company, then I build a couple a Website for friends and family like www.uneseguros.com.co and www.finfix.co
